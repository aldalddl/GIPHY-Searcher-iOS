> GIPHY 밈을 검색하고 저장하는 앱

</br>   

## Requirements
* iOS 14.0+     

</br>

* UIKit
* UserDefaults
* MVC  
* REST API

</br>   

## API

</br>   

## Libraries
* SnapKit
